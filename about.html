概要

Microsoft Azure を基盤としたクラウドアーキテクチャ支援を中心に、大手製造業・金融・公共領域での提案・設計・PoCに従事。ソリューションアーキテクト／PMとして、実装とマネジメントの両面を担う。

C#／.NET／SQL Serverを用いたアプリ開発、マイクロサービス構成、AI・IoT・生成AIまで幅広く対応可能。PoCや構成検証を得意とし、現場目線での提案と動くプロトタイプ作成に強みを持つ。

Microsoft認定資格（AZ-104, 204, 305）を保有。クラウドセキュリティやID統合、Entra ID / DevOps / OpenAIを用いた現場導入支援の実績あり。

スキルセット

  • Microsoft Azure（IoT Hub／AKS／App Services／Functions／DevOps／MLOps）
  • C#／.NET／SQL Server／オブジェクト指向設計
  • 生成AI／OpenAI／Azure OpenAI／AIソリューション提案
  • PoC設計・開発（AI・IoT領域中心）
