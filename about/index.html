経歴

日系SIer、大手外資系ITコンサルティングファームにて、Microsoft Azure を基盤としたクラウドアーキテクチャ支援を中心に、大手製造業・金融・公共領域での提案・設計・PoCに従事。ソリューションアーキテクト／PMとして、実装とマネジメントの両面を担う。

C#／.NET／SQL Server を用いたアプリ開発、マイクロサービス構成、ML・IoT・生成 AI まで幅広く対応可能。PoCや構成検証を得意とし、現場目線での提案と動くプロトタイプ作成に強みを持つ。

Microsoft 認定エキスパート資格（AZ-104, 204, 305等）を保有。クラウドセキュリティや ID 統合、Entra ID／DevOps／OpenAI を用いた現場導入支援の実績あり。
